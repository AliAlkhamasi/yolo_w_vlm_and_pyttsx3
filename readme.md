# Voice-Controlled Vision Assistant

YOLO object detection with VLM, speech recognition and TTS.

## Setup
```bash
pip install -r requirements.txt
```

Add your OpenAI API key in the code.

## Run
```bash
python voice_vision.py
```

Say **"analyze"** to trigger VLM analysis, press **Q** to quit.

## Features
- Real-time YOLO detection with FPS
- Voice-controlled activation
- GPT-4o Vision scene analysis
- Text-to-speech responses
